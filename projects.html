Lab 1: Web-Connected Thermometer

About this lab

In the Web-Connected Thermometer project, our objective was to create a sophisticated temperature monitoring system that offers both real-time data visualization and user-friendly interaction. At the headt of hour device is the Raspberry Pi Pico (Pico W), complemented by the high-resolution Pimoroni Display (PIM543) and the prevision-driver DS18S20 1-Wire Digital Thermometer. To enhance user experience and provide remote access capabilities, we devloped a web interface using React, with the coding environment facilitated by Visual Studio Code. The software foundation for this project was built upon CircuitPython for the hardware components and JavaScript for the web interface.

Lab 2: Electric Eye Light Beam-Interrupter Safety System

About the lab

The purpose of the electric eye light beam-interrupter safety system lab is to design, develop, and test a novel safety system aimed at reliably detecting, and responding to physical obstructions in a given environment. This project addresses a critical real-world problem wherein accurate and dependable sensor data are essential for ensuring the safety and functionality of various aplications. The system consists of a transmitter, which emits a modulated infrared (IR) beam, and a receiver, which detects interruptions in the beam and triggers an alert. Through a comprehensive exploration of design choices, experimentation, and testing, this report provides insights into the development of a safety system that performs effectively under diverse lighting conditions, while also incorporating an SMS alert feature.
